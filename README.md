#  Python para Iniciantes: Fundamentos e Aplicações Práticas

##  Contexto
Este projeto foi desenvolvido como um caderno de estudos com foco no aprendizado da linguagem Python, utilizando Inteligência Artificial como apoio para aprofundamento e organização do conhecimento.

O objetivo é consolidar os fundamentos da programação e explorar aplicações práticas que possam ser utilizadas no dia a dia profissional.

---

##  Objetivos

- Aprender os fundamentos da linguagem Python
- Desenvolver lógica de programação
- Criar pequenos scripts práticos
- Utilizar IA como ferramenta de apoio ao aprendizado
- Aplicar conceitos em situações reais

---

##  Conteúdos Estudados

- Introdução ao Python
- Variáveis e tipos de dados
- Estruturas condicionais (if/else)
- Estruturas de repetição (for/while)
- Funções
- Aplicações práticas

---

##  Engenharia de Prompts

Durante o desenvolvimento deste projeto, foram utilizados prompts estratégicos para extrair conhecimento da IA.

### Exemplos de Prompts:
- "Explique Python como se eu fosse iniciante absoluto"
- "Crie exemplos simples de código em Python com explicação"
- "Explique variáveis de forma prática"
- "Crie exercícios de Python para iniciantes com resposta"

### Dificuldades encontradas:
- Respostas muito genéricas
- Falta de exemplos práticos
- Necessidade de refinar perguntas

### Ajustes realizados:
- Uso de prompts mais específicos
- Solicitação de exemplos reais
- Pedidos de explicação passo a passo
