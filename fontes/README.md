#  Curadoria de Fontes

https://docs.python.org/pt-br/3/

Descrição:
Documentação oficial da linguagem Python, contendo conceitos, exemplos e guias completos.

https://www.cursoemvideo.com/

Curso gratuito e didático voltado para iniciantes, com explicações simples e práticas.
