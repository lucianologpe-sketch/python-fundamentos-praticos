#  Glossário de Termos em Python

Este glossário reúne os principais conceitos aprendidos durante o estudo de Python.

---

##  Python
Linguagem de programação de alto nível, simples e muito utilizada para diversas aplicações.

---

##  Variável
Espaço na memória usado para armazenar dados.

Exemplo:

    nome = "Luciano"

---

##  Tipo de Dado
Define o tipo de informação armazenada em uma variável.

Principais tipos:
- str (texto)
- int (número inteiro)
- float (número decimal)
- bool (verdadeiro ou falso)

---

##  Estrutura Condicional
Permite que o programa tome decisões com base em condições.

Exemplo:

    if idade >= 18:
        print("Maior de idade")

---

##  Loop (Laço de Repetição)
Estrutura usada para repetir ações.

Exemplo:

    for i in range(5):
        print(i)

---

##  Função
Bloco de código reutilizável que executa uma tarefa específica.

Exemplo:

    def saudacao():
        print("Olá!")

---

##  Algoritmo
Sequência de passos para resolver um problema.

---

##  Script
Programa simples escrito em Python para executar uma tarefa.

---

##  Lógica de Programação
Forma de organizar o pensamento para resolver problemas através de código.
