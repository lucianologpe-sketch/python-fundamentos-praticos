#  Miniguia de Estudo — Python para Iniciantes

Este miniguia reúne os principais aprendizados sobre Python de forma resumida, prática e reutilizável.

---

##  Resumo Geral

Python é uma linguagem de programação simples, versátil e muito utilizada no mercado.

Principais pontos aprendidos:

- Uso de variáveis para armazenar dados  
- Tipos de dados (texto, números e booleano)  
- Estruturas condicionais (if/else)  
- Estruturas de repetição (for)  
- Criação de funções  
- Aplicações práticas no dia a dia  

---

##  Glossário Essencial

- Variável: espaço para armazenar dados  
- Loop: repetição de ações  
- Função: bloco de código reutilizável  
- Condicional: tomada de decisão no código  
- Algoritmo: sequência lógica para resolver um problema

##  Aplicação Prática

Exemplo simples aplicado à logística:

    distancia = 150
    custo_por_km = 3

    frete = distancia * custo_por_km

    print("Valor do frete:", frete)

Objetivo:
Demonstrar como Python pode ser usado para cálculos simples no dia a dia profissional.
